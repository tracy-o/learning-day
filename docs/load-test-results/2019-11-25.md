# Load tests empty response Lambda with XRAY

- 25th November 2019
- Belfrage Test using Playground empty-response lambda
- No caching
- Vegeta Runner
- Tests run using http1
- XRAY enabled

### Instance

- Type: 1 x c5.2xlarge
- CPUs: 8 vCPUs (4 core, 2 threads per core)

### Lambda

- Memory: 512mb
